5.4 Extreme value theorem¶

To find absolute extrema for a function 𝑓 on a closed interval [𝑎,𝑏]:

  • Find all critical numbers for 𝑓 in (𝑎,𝑏)
  • Evaluate 𝑓 for all critical numbers in (𝑎,𝑏)
  • Evaluate 𝑓 at the end points 𝑎 and 𝑏
  • The largest value found in step 2 or 3 is the absolute maximum for 𝑓 on [𝑎,𝑏] and the smallest value found is the absolute minimum for 𝑓 on [𝑎,𝑏]

5.5 Horizontal asymptotes¶

The value of y=L where L is the value of function at x = +oo and -oo.

(We also have vertical and oblique asymptotes for some function)

In [32]:
# Let us find horizontal asymptotes of a given function
x = sym.symbols('x')
f = (2*x**2 + 1)/x**2

print('The given expression :')
display(f)

# Let us simplify the given expression
f = sym.simplify(f)

print('The simplified expression :')
display(f)

#Find and print the horizontal asymptotes
pha = sym.limit(f,x,sym.oo)
nha = sym.limit(f,x,-sym.oo)

print('The horizontal asymtots of the function at +oo is {} and -oo is {}'.format(pha,nha))

The given expression :
$\displaystyle \frac{2 x^{2} + 1}{x^{2}}$ 
The simplified expression :
$\displaystyle 2 + \frac{1}{x^{2}}$ 
The horizontal asymtots of the function at +oo is 2 and -oo is 2
